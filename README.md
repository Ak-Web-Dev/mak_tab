Your New Browser New Tab!
<h1>Introducing MAK TAB!</h1>
<p>A New Tab page for you Web Browser that is NASA-Inspired and has all the stuff you need. Includes a Calculator Too!</p>
<p><a href="https://ak-web-dev.github.io/mak_tab">https://ak-web-dev.github.io/mak_tab/</a></p>
<hr />
<h3><u>Key Features:</u></h3>
<p>1: NASA Image API Handler using JS</p>
<p>2: GoogleSans Font used</p>
<p>3: Bootsrap UI used</p>
<p>4: AOS Animation included</p>
<p>5: Working Local Time Clock</p>
<p>6: Working local weather temperature (requires Postal Code)</p>
<p>7: Calculator in the Tools Box (triggered with the Tools Button)</p>
<p>8: 5 Shortcut Buttons to your favourite websites which save the url using localStorage.setItem(‘’) in JS!</p>
<p>9: From 4 Welcome phrases, one is chosen randomly from each time you reload or open the page to give a living touch!</p>
<p>10: Small "Created by" Badge added cleanly.</p>
<hr />
<h3><u>How To Set it as your NewTab?</u></h3>
<p>If you are on <b>chrome or chromium-based</b> browser, download “New Tab Redirect” Extension from chrome web store and set the URL of this site to show as your Default New Tab!</p><br/>
<p>For <b>Safari:</b> Go to Safari > Settings > General (or press Cmd + , on Mac). In the Homepage field, enter the URL of the website you want to use. In the "New tabs open with" drop-down menu, select Homepage.</p>
<p>For <b>FireFox:</b> Click the menu button (three horizontal lines) in the top-right corner of Firefox. Select Add-ons and themes. Search for the New Tab Override or Custom New Tab Page extension. Click Add to Firefox and confirm the installation. Once installed, open the extension’s Options or Settings (you can usually click its icon near your address bar). Type or paste the URL (e.g., https://google.com) or select the file you want to use.
